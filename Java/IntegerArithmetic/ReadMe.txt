Instruction to run the project:
----------------------------------------------------------------------------

1) Unzip the folder.
2) Make sure the folder name is same as the package name.(i.e. IntegerArithmetic).
3) Compile both the codes(either both simultaneously or seperately using: 
			javac IntegerArithmetic/*.java
			javac IntegerArithmetic/TestLP1.java

4) Type following commands to run the different test cases from TestLP1.java
		Command: java -Xss512m -Xms2g IntegerArithmetic/TestLP1 0
			 java -Xss512m -Xms2g IntegerArithmetic/TestLP1 1
			 java -Xss512m -Xms2g IntegerArithmetic/TestLP1 2
			 java -Xss512m -Xms2g IntegerArithmetic/TestLP1 3
			 java -Xss512m -Xms2g IntegerArithmetic/TestLP1 4
			 java -Xss512m -Xms2g IntegerArithmetic/TestLP1 5
			 java -Xss512m -Xms2g IntegerArithmetic/TestLP1 6
